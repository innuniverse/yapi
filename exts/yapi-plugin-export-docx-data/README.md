# YAPI导出word文件插件
在平台”数据导出“下拉菜单增加”docx“选项，导出word文件
# 方案
使用[docxtemplater](https://docxtemplater.readthedocs.io/en/latest/#)，提供模板文件渲染数据，生成word文件
